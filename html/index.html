Why this repo exists

Visual summary

  • Creates the shared backend every later Terraform repo uses to store state.
  • Adds DynamoDB locking so concurrent plans and applies do not corrupt state.
  • Creates GitHub OIDC roles that let CI workflows deploy without static AWS keys.
